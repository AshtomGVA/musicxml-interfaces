Interface Lyric

Text underlays for lyrics, based on Humdrum with support for other formats.

IMPORTANT: is fake. All children of lyric-parts are actually children of lyric. This is a construct invented by musicxml-interfaces for separating ordered and unordered content.

Language names for text elements come from ISO 639, with optional country subcodes from ISO 3166. muiscxml-interfaces currently ignores this field.

Justification is center by default; placement is below by default. The print-object attribute can override a note's print-lyric attribute in cases where only some lyrics on a note are printed, as when lyrics for later verses are printed in a block of text rather than with each note.
